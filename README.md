# Core Rule Set migration
This RPM simplifies the migration process from the CRS v3.x detection rule set to CRS v4.24.0.

This RPM installs a modified version of the CRS v4.24.0 rule set. This prevents new rules and rules that have undergone significant changes from accumulating scores to determine blocking. Instead, these scores accumulate in an independent variable that can be monitored. Monitoring the decrease in this variable provides insight into when it's safe to install the new CRS version without so many false positives.

The rule for monitoring this variable is also generated by this RPM.
